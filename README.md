# Code Champs

## これはなに？

コードゴルフ用のレポジトリ。

## 仕様

* 「管理者」は簡単な「お題」を入稿管理することができる。
* お題は「期間」と「お題の文章」と「サンプルコード」と提出されたコードが正しいかをテストするための「テストケース」で構成される
* 「期間」は月単位で、一つの「お題」が有効な期間中は、他の「お題」は重複して登録できない
* 「参加者」は「お題」の「期間内」であれば、自由に「コード」を提出することができ、「テストケース」を全部passしたら文字数によって「スコア」が算出される
  * 「スコア」は「コメント」「空白文字」「改行」「PHPタグ」などを除外した「コードの文字数」がスコアになります。
  * 期間中に提出された「コード」は「スコア」の昇順でランキングされる、ただし同率の場合は提出時間の早いほうがランキング上位となる。
* 「お題」への提出言語は現在PHPのみ対応しているが、今後TSなども対応する可能性あり（ランキングは言語毎のランキングとなる想定）。
* 「期間内」は「参加者」同士お互いの「スコア」だけ見ることが出来、「期間終了後」は他の「参加者」の「コード」を見ることが出来ます。

## ミドルウェア構成

* Nginx
* PHP-FPM 8.4
  * Laravel 12
* MySQL 8.4

  * src/        Laravelプロジェクトファイル
* database/
  * conf        databaseの設定ファイル
* nginx/
  * conf/       nginxの設定ファイル
* Makefile      プロジェクトのsetupやupなどの操作のラッパー
* compose.yml   compose用のファイル
* .env          compose用の.envファイル
