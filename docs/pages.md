# ページ仕様

## 認証関連

### GET /register
- **概要**: ユーザー登録ページ表示
- **表示内容**: 
  - ユーザー名入力フォーム
  - メールアドレス入力フォーム
  - パスワード入力フォーム
  - 登録ボタン
- **認証**: 不要

### POST /register
- **概要**: ユーザー登録処理
- **フォームデータ**:
  - `name`: ユーザー名
  - `email`: メールアドレス
  - `password`: パスワード
- **成功時**: `/challenges` にリダイレクト
- **エラー時**: `/register` にリダイレクト（バリデーションエラー表示）

### GET /login
- **概要**: ログインページ表示
- **表示内容**:
  - メールアドレス入力フォーム
  - ログインボタン（メールリンク送信）
- **認証**: 不要

### POST /login
- **概要**: ログイン処理（メール認証リンク送信）
- **フォームデータ**:
  - `email`: メールアドレス
- **成功時**: メッセージ表示（認証リンクをメールに送信）
- **エラー時**: `/login` にリダイレクト（エラーメッセージ表示）

### GET /
- **概要**: トップページ
- **動作**:
  - ログイン済み: `/challenges` にリダイレクト
  - 未ログイン: `/login` にリダイレクト

## お題（チャレンジ）関連

### GET /challenges
- **概要**: お題一覧ページ表示
- **表示内容**:
  - お題のリスト（タイトル、説明、対応言語）
  - 各お題の詳細ページへのリンク
  - 管理者の場合：お題作成ボタン
- **認証**: 必須（未ログインの場合は `/login` にリダイレクト）

### GET /challenges/create
- **概要**: お題作成ページ表示
- **表示内容**:
  - タイトル入力フォーム
  - 説明入力フォーム
  - 期間設定フォーム（開始日、終了日）
  - 対応言語とサンプルコード設定
  - 作成ボタン
- **認証**: 管理者のみ

### POST /challenges
- **概要**: お題作成処理
- **フォームデータ**:
  - `title`: タイトル
  - `description`: 説明
  - `period_start`: 開始日 (YYYY-MM-DD)
  - `period_end`: 終了日 (YYYY-MM-DD)
  - `languages[n][language_id]`: 言語ID
  - `languages[n][sample_code]`: サンプルコード
- **成功時**: `/challenges` にリダイレクト（成功メッセージ表示）
- **エラー時**: `/challenges/create` にリダイレクト（バリデーションエラー表示）
- **認証**: 管理者のみ

### GET /challenge/{id}
- **概要**: お題詳細ページ表示
- **表示内容**:
  - お題のタイトル、説明
  - 対応言語リスト
  - 言語選択とサンプルコード
  - コード入力エディタ
  - 提出ボタン
  - ランキングページへのリンク
  - 管理者の場合：編集・削除ボタン
- **認証**: 必須

### GET /challenges/{id}/edit
- **概要**: お題編集ページ表示
- **表示内容**:
  - お題情報編集フォーム（タイトル、説明、期間など）
  - 更新ボタン
- **認証**: 管理者のみ

### PUT /challenges/{id}
- **概要**: お題更新処理
- **フォームデータ**:
  - `title`: タイトル
  - `description`: 説明
  - `period_start`: 開始日
  - `period_end`: 終了日
  - その他更新項目
- **成功時**: `/challenge/{id}` にリダイレクト（成功メッセージ表示）
- **エラー時**: `/challenges/{id}/edit` にリダイレクト（バリデーションエラー表示）
- **認証**: 管理者のみ

### DELETE /challenges/{id}
- **概要**: お題削除処理
- **成功時**: `/challenges` にリダイレクト（成功メッセージ表示）
- **エラー時**: エラーメッセージ表示
- **認証**: 管理者のみ

### GET /challenges/{id}/ranking
- **概要**: お題のランキングページ表示
- **表示内容**:
  - 提出者のランキングリスト（順位、ユーザー名、スコア、提出日時）
  - 提出期間終了後：各提出者のコード閲覧リンク
- **認証**: 必須

## 提出（サブミッション）関連

### POST /submissions
- **概要**: コード提出処理
- **フォームデータ**:
  - `challenge_id`: お題ID
  - `language_id`: 言語ID
  - `code`: 提出コード
- **成功時**: `/challenge/{challenge_id}` にリダイレクト（提出結果表示）
- **エラー時**: `/challenge/{challenge_id}` にリダイレクト（エラーメッセージ表示）
- **認証**: 必須

### GET /challenges/{id}/submissions/{submission_id}
- **概要**: 提出コード詳細表示
- **表示内容**:
  - 提出されたコード
  - 実行結果
  - スコア
  - 提出日時
- **表示条件**: 
  - 自分の提出：常に閲覧可能
  - 他人の提出：お題の提出期間終了後のみ閲覧可能
- **認証**: 必須

## 共通動作

### エラーハンドリング
- **バリデーションエラー**: フォームに戻り、エラーメッセージを表示
- **認証エラー**: `/login` にリダイレクト
- **認可エラー**: 403 Forbiddenページまたは元のページにリダイレクト（エラーメッセージ表示）
- **404エラー**: 404ページ表示

### フラッシュメッセージ
- **成功**: 緑色の成功メッセージ表示
- **エラー**: 赤色のエラーメッセージ表示
- **情報**: 青色の情報メッセージ表示
